# DNA 序列分析工具插件

提供便捷的DNA序列基础分析功能，支持格式化、反向互补、翻译、GC含量/重复序列检测、引物Tm值计算及常见酶切位点查找。

---

## ⚒ 功能特性

- 🧬 **序列格式化**：按指定宽度（默认60 bp）分行输出，便于查看
- 🔄 **反向互补**：一键计算DNA序列的反向互补链
- 🧪 **翻译**：使用标准遗传密码表将DNA翻译为氨基酸序列
- 📊 **GC含量 & 重复序列**：计算GC百分比，并自动查找长度2–6 bp的简单重复序列
- 🌡️ **引物分析**：基于碱基组成估算熔解温度（Tm值），适合短引物（<14 bp）
- 🔬 **酶切位点查找**：内置常见限制性内切酶（EcoRI、BamHI、HindIII等）识别位点
- 🔐 **群组白名单**：支持配置允许使用该插件的群号列表

## ⌨️ 安装与配置

### 1. 安装插件

将插件文件 `dna_tools.py` 放入 AstrBot 的 `addons` 目录（或根据您的插件管理方式放置），然后重启 AstrBot 或通过 WebUI 重载插件。

### 2. 配置文件（可选）

在 AstrBot WebUI 中配置以下参数（或手动编辑 `data/star/dna_tools/_conf_schema.json` 对应项）：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `group_whitelist` | list | `[]` | 允许使用DNA工具的群号列表，为空则所有群可用。 |
| `format_width` | int | `60` | 序列格式化时每行的碱基数。 |

#### 配置示例（`_conf_schema.json`）
```json
{
    "group_whitelist": {
        "description": "允许使用DNA工具的群号列表，留空表示所有群",
        "type": "list",
        "default": []
    },
    "format_width": {
        "description": "序列格式化时每行的碱基数",
        "type": "int",
        "default": 60
    }
}
```
## 📋 命令说明

所有命令均以 **DNA** 或 **dna** 开头，后跟空格分隔的子命令和序列参数。

| 命令（英文 / 中文） | 用法 | 说明 |
|------|------|------|
| `format` / `格式化` | `DNA format <序列>` | 将序列按配置宽度（默认60 bp）分行输出。 |
| `reverse` / `反向互补` | `DNA reverse <序列>` | 输出序列的反向互补链。 |
| `translate` / `翻译` | `DNA translate <序列>` | 使用标准遗传密码表将DNA翻译为氨基酸序列（遇终止密码子显示 `*`，未知密码子显示 `X`）。 |
| `gc` / `gc含量` | `DNA gc <序列>` | 计算GC百分比，并自动查找长度2–6 bp的简单重复序列（最多显示前5个）。 |
| `primer` / `引物分析` | `DNA primer <序列>` | 计算GC%和Tm值（适用于短引物，公式：Tm = 4×(G+C) + 2×(A+T)）。 |
| `restriction` / `酶切` | `DNA restriction <序列>` | 查找内置常见限制酶（EcoRI、BamHI、HindIII等）的识别位点，返回酶名、识别序列及起始位置。 |
| `help` / `帮助` | `DNA help` | 显示完整的帮助信息。 |

> **序列参数规则**：  
> - 序列中的空格、换行会被自动忽略。  
> - 不区分大小写，只允许字母 `A`、`T`、`C`、`G`、`U`、`N（即A、T、C、G 中的任何一种）`。  
> - 若序列包含非法字符，插件会提示错误。

### 💡 使用示例
```
DNA format ATCGATCGATCGATCG

DNA 反向互补 ATGC

DNA translate ATGCCGTAA

DNA gc ATGCGCAT

DNA primer ATGCGT

DNA 酶切 GAATTC

DNA help
```

## ⌛ 工作流程

- 用户发送 `DNA` 命令并指定子命令及序列。
- 插件校验序列合法性，随后调用对应功能函数进行计算。
- 计算结果以文本形式返回给发送者（当前会话）。

## ‼️ 注意事项

- **翻译**：忽略末尾不完整的密码子，从第一个碱基开始每3个一组翻译；若序列长度不足3，则不输出任何氨基酸。
- **Tm值公式**：仅适用于长度短于14 bp的引物，更长序列建议使用热力学算法。
- **酶切位点**：目前内置10种常见限制酶（EcoRI、BamHI、HindIII、NotI、XbaI、SalI、PstI、SmaI、KpnI、SacI），如需更多可自行修改代码中的 `enzymes` 字典。
- **重复序列检测**：仅查找2~6 bp的简单重复（如 `ATAT`、`GCA`），不检测复杂重复或串联重复。
- **白名单**：若配置了 `group_whitelist`，非白名单群中的命令会被拒绝；私聊不受白名单限制。

## ⁉️ 常见问题

- **为什么翻译结果中出现 `*`？**  
表示终止密码子（TAA、TAG、TGA），蛋白质翻译结束。

- **为什么有些密码子翻译为 `X`？**  
该密码子不在标准遗传密码表中，通常因序列包含N或其他非ATCG字符，或读取框错误。

- **GC含量与重复序列输出内容太长？**  
重复序列列表仅显示前5个，如需查看全部可修改代码中的切片数量。

- **如何增加新的酶切位点？**  
编辑 `find_restriction_sites` 方法中的 `enzymes` 字典，添加 `'酶名': '识别序列'` 即可。

- **能否支持RNA序列？**  
当前仅针对DNA，但您可以用 `U` 替代 `T`，反向互补时会正确处理U→A。

- **是否支持文件输入或长序列？**  
目前仅支持命令行直接输入，对于很长的序列建议通过其他方式传输文件。
